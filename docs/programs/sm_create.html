sm_create¶

“Clone” a ShakeMap from NEIC Comcat, or create an event from scratch.

Notes on usage:

eventid is a ComCat event ID. For example, for this event: https://earthquake.usgs.gov/earthquakes/eventpage/us2000ar20 The event ID is us2000ar20.

If no source is specified, then the event ID used for the event directory, eventid field in event.xml file, and names of data and fault files will be that of the authoritative origin.

If a source (us, ci, nc, etc.) is specified, then that ID is used instead of the authoritative ID.

usage: sm_create [-h] [-f] [-s SOURCE] [-e NETID TIME LON LAT DEP MAG LOCSTR]
                 [--network NETWORK] [-b] [-n] [-p]
                 eventid

Positional Arguments¶

eventid ID of the event to process

Named Arguments¶

-f, --force Force overwrite of event data.
-s, --source Specify the source network of desired shakemap.
-e, --event Specify the event parameters (locstr should be in quotes)
--network Specify network name to be filled in event.xml file
-b, --skip-bounds
 Skip bounds set in online shakemap.
-n, --no-scenario
 When used with -e, disable scenario mode.
-p, --preserve-params
 Preserve model parameters detected in ComCat.
